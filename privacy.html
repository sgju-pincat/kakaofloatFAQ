개인정보처리방침

시행일: 2026-04-21

안읽몰톡(이하 “앱”)은 사용자의 개인정보를 중요하게 생각하며 관련 법령을 준수합니다. 본 방침은 앱이 어떤 정보를 어떤 목적으로 처리하는지 안내합니다.

1. 처리하는 정보

  • 알림 정보(선택): 사용자가 알림 접근 권한을 허용한 경우, 앱은 기기에 표시되는 알림의 제목, 내용, 발신 앱 정보 등을 기기 내에서 처리할 수 있습니다.
  • 음성 데이터(선택): 사용자가 음성 인식(STT) 기능을 사용하는 경우, 마이크를 통해 입력된 음성이 텍스트 변환을 위해 처리될 수 있습니다.
  • 위치 정보(선택): 사용자가 현위치 날씨 기능을 사용하는 경우, 현재 위치 좌표를 처리할 수 있습니다.
  • 위치 검색어(선택): 사용자가 날씨 위치 검색 기능을 사용하는 경우, 입력한 지역명 또는 검색어를 처리할 수 있습니다.
  • 앱 설정 정보: 사용자가 앱에서 선택한 설정값(예: 표시 옵션, UI 설정, 날씨 위치, 일부 캐시 정보 등)을 기기 내에 저장할 수 있습니다.
  • 결제 정보: 유료 기능(프로)을 제공하기 위해 Google Play 결제(인앱결제)를 사용합니다. 앱은 카드번호 등 결제수단 정보를 직접 수집하거나 저장하지 않습니다.
  • 화면 캡처 이미지(선택): 사용자가 “버블 꾹 눌러 버블 없는 화면 캡처” 기능을 켜고 직접 실행한 경우, 앱은 접근성 서비스를 통해 현재 화면 이미지를 캡처하여 사용자 기기 내 저장소(예: Pictures/Screenshots)에 저장할 수 있습니다. 캡처 이미지에는 사용자가 보고 있는 다른 앱의 화면 내용이 포함될 수 있습니다.
  • 접근성 서비스 정보(선택): 앱은 사용자가 명시적으로 활성화한 경우에만 접근성 서비스를 사용하며, 버블을 잠시 숨긴 뒤 사용자가 요청한 화면 캡처 기능을 수행하기 위해 사용합니다.
  • 문의 정보(선택): 사용자가 이메일 문의 기능을 이용하는 경우, 문의 내용과 함께 기기 정보, 앱 버전, 권한 상태, 구매 상태, 일부 설정 정보가 이메일 본문 또는 클립보드에 포함될 수 있습니다.

2. 처리 목적

  • 읽음 표시 없이 알림 기반으로 메시지 확인 기능 제공
  • 오버레이(버블) UI를 통한 빠른 확인 및 답장 기능 제공
  • 음성 인식(STT)을 통한 메시지 작성 기능 제공
  • 현위치 또는 선택 지역 기반 날씨 정보 제공
  • 유료 기능 제공 및 구매 상태 확인
  • 사용자가 직접 요청한 버블 없는 화면 캡처 기능 제공
  • 문의 대응, 오류 분석 및 고객지원

3. 데이터 저장 및 전송

  • 앱 설정 정보, 일부 알림 기반 기록 정보, 날씨 위치 정보 및 날씨 캐시 정보는 사용자 기기 내에 저장될 수 있습니다.
  • 앱은 사용자의 알림 내용, 메시지 내용, 채팅 내역을 개발자의 자체 서버로 전송하거나 판매하지 않습니다.
  • 단, 사용자가 STT 기능을 사용할 경우, 텍스트 변환 과정에서 Android 또는 Google 음성 인식 서비스에 의해 음성 데이터가 처리될 수 있습니다.
  • 사용자가 현위치 날씨 기능을 사용할 경우, 날씨 정보 제공을 위해 위치 좌표가 외부 날씨 서비스로 전송될 수 있습니다.
  • 사용자가 날씨 위치 검색 기능을 사용할 경우, 입력한 위치 검색어가 외부 지도/지오코딩 서비스로 전송될 수 있습니다.
  • 사용자가 이메일 문의 기능을 사용할 경우, 사용자가 직접 전송하는 이메일 본문에 포함된 정보가 사용자가 선택한 이메일 서비스 및 개발자 이메일 주소로 전달될 수 있습니다.
  • 사용자가 기기 정보 복사 기능을 사용할 경우, 해당 정보는 사용자의 기기 클립보드에 복사될 수 있습니다.
  • 화면 캡처 이미지는 사용자가 기능을 직접 실행한 경우 사용자 기기 내 저장소에 저장되며, 앱은 이를 개발자 서버로 전송하지 않습니다.
  • 캡처 이미지는 사용자가 직접 공유하거나 백업 앱, 갤러리 앱, 클라우드 앱 등을 통해 전송하지 않는 한 앱에 의해 외부로 전송되지 않습니다.
  • 접근성 서비스는 화면 캡처 기능 실행을 위해서만 사용되며, 화면 내용이나 입력 내용을 지속적으로 모니터링하거나 개발자 서버로 전송하지 않습니다.

4. 보관 및 파기

  • 사용자가 앱 데이터를 삭제하거나 앱을 삭제하면 기기 내 저장 정보는 함께 삭제될 수 있습니다.
  • 사용자가 개발자에게 보낸 문의 이메일은 문의 대응 및 서비스 개선을 위해 필요한 기간 동안 보관될 수 있으며, 목적 달성 후 삭제합니다. 다만 관련 법령에 따라 보관이 필요한 경우 해당 기간 동안 보관할 수 있습니다.

5. 제3자 제공 및 외부 서비스

  • 앱은 사용자의 개인정보를 제3자에게 판매하거나 임의로 제공하지 않습니다.
  • 인앱결제는 Google Play 결제 서비스를 통해 처리되며, 이 과정에서 Google이 구매 및 결제 관련 정보를 처리할 수 있습니다.
  • 현위치 확인은 Google 위치 서비스가 사용될 수 있습니다.
  • 날씨 정보 조회는 Open-Meteo 서비스를 이용할 수 있습니다.
  • 위치 검색은 OpenStreetMap Nominatim 서비스를 이용할 수 있습니다.
  • 이메일 문의는 사용자가 선택한 이메일 앱 또는 메일 서비스 제공자를 통해 전송됩니다.

6. 권한 안내

  • 알림 접근 권한: 메시지 알림을 앱 기능에 활용하기 위해 사용될 수 있습니다(사용자 동의 시).
  • 마이크 권한: STT 기능을 사용자가 실행할 때에만 음성 입력을 받기 위해 사용될 수 있습니다(사용자 동의 시).
  • 위치 권한: 현위치 날씨 기능 사용 시 현재 위치를 확인하기 위해 사용될 수 있습니다(사용자 동의 시).
  • 다른 앱 위에 표시(오버레이): 채팅 버블 등 화면 위 표시 기능 제공을 위해 사용될 수 있습니다(사용자 동의 시).
  • 사용자는 기기 설정에서 권한을 언제든지 변경하거나 철회할 수 있습니다.
  • 접근성 서비스(선택): 사용자가 “버블 없는 화면 캡처” 기능을 사용할 때, 버블을 잠시 숨기고 현재 화면을 캡처하기 위해 사용될 수 있습니다. 이 기능은 사용자가 직접 켠 경우에만 동작하며, 캡처 이미지는 기기 내에 저장됩니다.

7. 포그라운드 서비스 안내

앱은 오버레이 버블 유지, STT 기능, 일부 위치 기반 기능 제공을 위해 포그라운드 서비스를 사용할 수 있으며, 이 경우 사용자에게 지속 알림(ongoing notification)이 표시될 수 있습니다.

8. 이용자의 권리

사용자는 앱 설정 및 기기 설정을 통해 권한을 관리할 수 있으며, 앱 삭제를 통해 기기 내 저장된 정보의 삭제를 요청할 수 있습니다.

9. 아동 관련 안내

앱은 만 13세 미만 아동을 대상으로 하지 않으며, 만 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.
